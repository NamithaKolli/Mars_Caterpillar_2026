# Mars-Software-team-2025
all important files and folders for our rover 
## ROS 2 Topic Architecture

###  Perception & Sensors (RealSense D435i)
| Topic | Type | Description |
| :--- | :--- | :--- |
| `/camera/depth/color/points` | `sensor_msgs/PointCloud2` | Raw 3D PointCloud for obstacle avoidance |
| `/camera/realsense2_camera/color/image_raw` | `sensor_msgs/Image` | RGB Stream |
| `/camera/realsense2_camera/imu` | `sensor_msgs/Imu` | Combined Accel/Gyro data |
| `/camera/realsense2_camera/depth/image_rect_raw` | `sensor_msgs/Image` | Rectified Depth stream |

###  SLAM & Mapping (RTAB-Map)
| Topic | Type | Description |
| :--- | :--- | :--- |
| `/map` | `nav_msgs/OccupancyGrid` | 2D Static Map for Nav2 |
| `/cloud_map` | `sensor_msgs/PointCloud2` | Full 3D Map cloud |
| `/odom` | `nav_msgs/Odometry` | RTAB-Map optimized visual odometry |
| `/octomap_binary` | `octomap_msgs/Octomap` | Compressed 3D voxel map |
| `/rtabmap/imu` | `sensor_msgs/Imu` | IMU data filtered for SLAM |

###  Navigation Stack (Nav2)
| Topic | Type | Description |
| :--- | :--- | :--- |
| `/cmd_vel` | `geometry_msgs/Twist` | Final velocity commands to hardware |
| `/global_costmap/costmap` | `nav2_msgs/Costmap` | Global planning layer (Static Map based) |
| `/local_costmap/costmap` | `nav2_msgs/Costmap` | Local rolling window (Sensor based) |
| `/plan` | `nav_msgs/Path` | Global path from start to goal |
| `/local_path` | `nav_msgs/Path` | Local trajectory generated by Controller |
| `/goal_pose` | `geometry_msgs/PoseStamped` | Input target for the robot |

### Vision & Detections
| Topic | Type | Description |
| :--- | :--- | :--- |
| `/apriltag/detections` | `apriltag_msgs/AprilTagDetectionArray` | AprilTag coordinates |
| `/aruco/detections` | `ros2_aruco_interfaces/ArucoMarkers` | ArUco Marker info |
| `/landmark_detections` | `rtabmap_msgs/Landmarks` | Visual landmarks for SLAM loop closure |

### System & TF
| Topic | Type | Description |
| :--- | :--- | :--- |
| `/tf` | `tf2_msgs/TFMessage` | Dynamic transforms (`odom` -> `base_link`) |
| `/tf_static` | `tf2_msgs/TFMessage` | Static transforms (`base_link` -> `camera_link`) |
| `/diagnostics` | `diagnostic_msgs/DiagnosticArray` | Hardware and node status |
